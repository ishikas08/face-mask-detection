# Face Mask Detection (Deep Learning + OpenCV)
Dataset:
https://www.kaggle.com/datasets/ashishjangra27/face-mask-12k-images-dataset

Technologies Used:
Python
TensorFlow / Keras (CNN model)
OpenCV
Haar Cascade Face Detector
Kaggle Face Mask Dataset
Google Colab for training
Matplotlib for analysis

Features:
Real-time face mask detection from video
Custom trained model
Detection visualization (Mask vs No Mask)
Accuracy and loss graphs included

Model Architecture:
Image preprocessing
CNN layers
Binary classifier (Mask / No Mask)

Face Detection Approach:
Haar Cascade Classifier (haarcascade_frontalface_default.xml)

Results:
Works in real-time
Detects mask with high precision in good lighting
Annotated video output with bounding boxes
